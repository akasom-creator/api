# api

A simple NestJS API using Prisma for database access, ready for deployment.

## Getting Started

### Prerequisites

- Node.js (v16.13 or later)
- npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/akasom-creator/api.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the App

```bash
# development
npm run start:dev

# production mode
npm run start:prod
```
